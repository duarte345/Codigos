1º documento em HTML

A 1ª coisa que um principiante de construção de páginas HTML deve compreender é que uma página de hipertexto é uma página de texto normal, reforçado com uma série de caracteres especiais que permitem formatar o texto para apresentação no monitor.

Algo que por vezes confunde os principiantes é que, apesar de o código HTML num servidor ser o mesmo, é consultado por vários browsers e pode ser interpretado de forma diferente, consoante o browser utilizado.
